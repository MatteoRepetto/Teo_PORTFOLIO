THE GREEN RECIPE ARCHIVE

2023 // DATA VIZ - UI DESIGN - VISUAL IDENTITY - EDITORIAL DESIGN

Fast food started talking about environmental consciousness: by using a website as an official communication tool, they can at ease reach an audience and spread their green narratives. This strategy is known as “Greenwashing”. "The Green Recipe Archive" is structured as a collection of these green narratives, in which real examples of sentences extracted from their websites can be consulted and explored. The project includes a first part of data collection showcased in a report and a second part of data-viz with a website-archive.

Role
Visual identity and UI design, Data analysis, Editorial design, Web design

Softwares used
Figma, Visual studio code, Illustrator, Google Spreadsheet
