ROBBIN' THE CROWN

ASSETS DESIGN // ART DIRECTION // CHARACTER DESIGN // 3D MODELING

“Robbin‘ the Crown” is a platform video game, sequel to the famous Disney classic “Robin Hood” and visually inspired by "Night in the Woods". The game is developed on a single level and is self-concluding. In addition to the video game, 3D renderings have been created for the same project that tell the plot in detail.
