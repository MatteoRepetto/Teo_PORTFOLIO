SPIT IT OUT!

LOGO DESIGN // WEB DESIGN // WIREFRAME DESIGN // VISUAL IDENTITY

“Spit it Out!” site is a common space that presents a diary as collaborative canvas: people will create a masterpiece of “words traces”. Step by step, at the user will be asked some questions in order to generate a “river of words”. The final result is an artwork in which all the bad and good feelings dance together.
